# 互动小宠物 (磁流体乌咪) 使用说明

## 1. 如何运行本项目

本项目使用了 HTML5 Canvas、 SVG 融合滤镜和高级交互功能。为了保证所有功能（特别是局域网分享和 AI 手势识别）正常工作，必须通过**本地服务器**运行，而不是直接双击打开 HTML 文件。

**运行方法**：
在电脑终端（Terminal）中，进入本项目所在的文件夹，运行以下命令启动服务：
```bash
python3 -m http.server 8000
```
然后在电脑浏览器中打开网址：`http://localhost:8000`

---

## 2. 交互功能指南

* **🖱️ 鼠标互动**：将鼠标移入宠物的脑袋范围，黑色的磁流体会像被磁铁吸引一样，紧紧跟着你的鼠标指针移动。
* **💤 待机模式**：当你移开鼠标或停止互动超过 0.5 秒时，宠物会进入“发呆”状态，自动变形成一条起伏的横杠和一个小圆点，并伴随呼吸起伏。
* **✨ 点击惊喜**：
  * **单击**：有 50% 的概率聚集成一个“叹号”，50% 的概率炸开变成动态旋转的“刺球”。
  * **快速连击 3 次**：会强制触发隐藏的“猫猫嘴 (:3)”表情，并伴随 Q 弹的跳动！
* **🖐️ AI 摄像头手势**：点击网页右上角的“⚙️ 设置”，选择“📷 开启手势互动”。允许浏览器调用摄像头后，你可以隔空用**食指指尖**来引导磁流体！屏幕上会出现一个小绿点作为你的手势光标反馈。

---

## 3. 手机 / iPad 访问指南

你可以用手机或 iPad 上的浏览器打开这个网页，用手指触摸屏幕来“撸猫”。

**操作步骤**：
1. 确保你的手机/iPad 与这台运行网页的电脑连接在 **同一个 Wi-Fi（同一局域网）** 下。
2. 在电脑上打开本网页，点击右上角“⚙️ 设置”。
3. 在面板底部，你会看到 **“📱 手机/iPad 访问”** 的提示。
   * *如果你是在 `localhost` 下打开的，它会提示你点击一个 `192.168.X.X` 开头的局域网 IP 链接。*
   * *点击该链接后，设置面板里会自动生成一个**二维码**！*
4. 用手机相机直接扫码，或者在手机浏览器中输入该局域网 IP 地址，即可开始互动。

> **⚠️ 关于手机摄像头的特别说明**：
> 由于苹果 iOS 和安卓系统的严格安全限制，手机端通过局域网 HTTP 协议访问网页时，浏览器会拒绝提供摄像头权限（必须是 HTTPS 协议才行）。
> 因此，手机端**无法使用 AI 手势识别功能**，但你依然可以用手指直接在屏幕上滑动、点击来和宠物进行物理互动！
